# Echo 项目经验日志 — 5月16日

## 🎯 今日完成

### 1. 语音消息播放 (VoiceBubble)
- **问题**: 语音消息只显示时长文本 "5s"，无法播放
- **解决**: 创建 VoiceBubble 组件，内嵌 HTML5 `<audio>` 元素，点击切换播放/暂停，实时显示进度条+波形动画，音频结束自动复位
- **文件**: `frontend/src/components/VoiceBubble.tsx`

### 2. 消息搜索高亮 + 跳转
- **问题**: 搜索结果无高亮，点击无反应
- **解决**: 正则 split + `<mark>` 标签实现黄色高亮；点击搜索结果跳转到 `/chat/:peerId` 对应聊天
- **文件**: `frontend/src/pages/SearchMessagesPage.tsx`

### 3. FloatingStackGallery 悬浮循环堆叠画廊
- **核心**: Absolute 绝对定位 + Framer Motion spring + 10秒自动倒放
- **堆叠**: 第1张 z-20 scale(1)，第2张 z-10 scale(0.95) y+12，第3张 z-0 scale(0.9) y+24
- **左滑切图**: drag="x" + dragConstraints + onDragEnd 检测 offset.x < -50
- **10秒倒放**: useEffect 监听 offset，setTimeout 10s → setInterval 250ms 逐步回退到 0
- **文件**: `frontend/src/components/FloatingStackGallery.tsx`

### 4. PaginatedGridGallery 宫格横滑分页
- **布局**: grid-cols-3，每页最多9张
- **分页**: 数组 slice 成 pages，motion.div animate x: -page*100%
- **手势**: drag="x" + dragElastic=0.2 + onDragEnd 判断方向
- **文件**: `frontend/src/components/PaginatedGridGallery.tsx`

### 5. 画廊接入星轨
- MomentsPage: <=3 张用 FloatingStack，>=4 张用 PaginatedGrid
- PersonalOrbitView: 多图用 FloatingStackGallery

## 🐛 踩坑记录

### 坑1: JSX 结构错误导致 TypeScript 编译失败
- **现象**: `Expected corresponding JSX closing tag for 'motion.div'`
- **原因**: 用 motion.div 替换 div 时，忘记同步修改闭合标签
- **解决**: 仔细检查开闭标签配对，确保 `</motion.div>` 对应 `<motion.div>`

### 坑2: macOS/Linux 换行符警告
- **现象**: `LF will be replaced by CRLF`
- **原因**: Windows Git 默认配置
- **影响**: 不影响功能，仅编译时警告

### 坑3: tar 打包路径错误
- **现象**: 解压后缺少 docker-compose.prod.yml
- **原因**: `cd /d/临时/Task/Echo` 在 bash 中不生效
- **解决**: 使用 `pwd` 确认当前目录后再 tar
