## 防内存溢出 (OOM) 管线

手机原图可达 8-12MB，直接上传会导致浏览器内存溢出和请求超时。

### 前端压缩方案
```typescript
import imageCompression from 'browser-image-compression';

export async function compressImage(file: File): Promise<File> {
  return await imageCompression(file, {
    maxSizeMB: 0.3,           // 最大 300KB
    maxWidthOrHeight: 1600,   // 限制分辨率
    useWebWorker: true,       // 不阻塞主线程
    fileType: 'image/webp',   // WebP 格式
    initialQuality: 0.7,      // 初始质量
  });
}
```

效果：8MB → 200KB，上传速度提升 40 倍。

## 移动端键盘避让方案

传统 `h-screen` 在键盘弹出时布局崩溃，出现半屏黑屏。

### 解决方案
```html
<div className="min-h-[100dvh] flex flex-col overflow-y-auto">
```

- `dvh` (dynamic viewport height) 随键盘自动调整
- `overflow-y-auto` 内部滚动
- `pb-[env(safe-area-inset-bottom)]` 安全区适配

## 乐观更新 (Optimistic UI)

在弱网或 VPN 环境下，发送消息立即本地渲染：

```typescript
// 1. 立即创建临时消息
const tempId = 'temp-' + Date.now();
const optimisticMsg = { ... };
setMessages(prev => [...prev, optimisticMsg]);

// 2. 异步发送
socket.emit('message:send', ..., (res) => {
  if (res?.error) {
    // 3. 失败标记红点
    setMessages(prev => prev.map(m => m.id === tempId ? {...m, content: m.content + ' ❌'} : m));
  } else {
    // 4. 替换为真实消息
    setMessages(prev => prev.map(m => m.id === tempId ? res.message : m));
  }
});
```
