# Echo 篇 2：引力圈 — 在聊天软件里造一个黑洞

## 概念

引力圈（Gravity Zone）是 Echo 的核心创新——一个隐藏的"重要联系人星系"。用户可以从喧闹的会话列表中把最重要的联系人"捕获"至此。

打开方式：在首页向左滑动（带弹簧物理阻尼），进入全屏沉浸的引力圈页面。右滑退出。

## 视觉设计

**深色模式**：OLED 纯黑底色 + 径向渐变深空背景，正中央悬浮高能等离子吸积盘黑洞。双层 conic-gradient 反向旋转 + drop-shadow 辉光 + 事件视界核心拉伸动画。

**浅色模式**：极简银灰 `#f8fafc` 太空舱背景。黑洞视觉效果在两种模式下保持一致。

**星球卡片**：毛玻璃 `backdrop-blur-xl` + 内部 blob 浮动 + 引力锁定标识。长按可移出引力圈。

## 核心技术点

### 手势锁机制

引力圈卡片长按会触发 300ms 锁。在锁期间，所有右滑退出事件被拦截，防止长按松手误触返回。

```typescript
const lock = () => { lockRef.current = true; setTimeout(() => { lockRef.current = false; }, 300); };
// 右滑退出时检查: if (dx > 40 && !lockRef.current) onClose();
```

### 侧边栏阻断

引力圈打开时，全局侧边栏唤出手势必须被禁用。在 ChatPage 的 touchEnd 中加判断：

```typescript
if ((paramId && showChatMobile) || showGravity) return;
```

### 导航记忆

从引力圈进入聊天后，返回时自动回到引力圈而非首页。通过 `localStorage` 标记实现：

```typescript
localStorage.setItem('echo-gravity-return', '1');
```

## 踩坑记录

**深色光晕穿透**：`radial-gradient` 写死浅色终点导致白色光圈。改为 JS 条件 `style={{ background: isDark ? '...' : '...' }}`。

**吸积盘太暗**：initial 版本用了低透明度 `rgba(..., 0.x)`，几乎看不见。改为高饱和纯色 + double `drop-shadow` + `mix-blend-mode: color-dodge` 才达到高能等离子体效果。
