# Echo 稀土掘金篇 2：引力圈与手势交互引擎

## 引力圈 (Gravity Zone) 的设计哲学

引力圈是 Echo 的核心创新——一个隐藏在首页左侧的"重要联系人星系"。用户可以将最重要的联系人从喧闹的会话列表中"捕获"至此。

### 视觉设计

- **浅色模式**: 径向渐变 `radial-gradient(circle at center, rgba(99,102,241,0.05) → #f9fafb)`
- **深色模式**: 纯黑 `#05050A`，正中央悬浮呼吸动画星球 🪐
- **轨道环**: 3层同心圆 `border-indigo-400/purple-400/violet-400 opacity-[0.03]`

### 核心：手势交互引擎

Echo 的手势系统是整个应用流畅体验的基石。以下是完整的手势判定逻辑：

```typescript
// 首页左滑 → 进入引力圈
ConversationList touchEnd: dx < -50 → e.stopPropagation() + onOpenGravity()

// 引力圈右滑 → 回首页
GravityZone touchEnd: dx > 40 + !lockRef.current → onClose()

// 侧边栏：左边缘(x < 屏幕20%) + 右滑(dx > 40)
ChatPage touchEnd: edgeX < width*0.20 && dx > 40 → open sidebar

// 聊天室：右滑退出，左滑个人星轨
ChatWindow touchEnd: dx > 50 → onBack() | dx < -50 → showPersonalOrbit
```

### 防误触锁机制

引力圈卡片长按会触发 300ms 锁：
```typescript
const lock = () => { lockRef.current = true; setTimeout(() => { lockRef.current = false; }, 300); };
```
在锁期间，所有右滑退出事件被拦截，防止长按松手误触返回。

### 手势冲突解决方案

| 场景 | 冲突 | 解决 |
|------|------|------|
| 左滑进引力圈 vs 侧边栏弹出 | 两者同时触发 | stopPropagation + 严格坐标阈值 |
| 聊天中右滑 vs 侧边栏 | 穿透 | `paramId && showChatMobile` 时阻断外层手势 |
| 引力圈卡片长按 vs 原生菜单 | 系统菜单弹出 | `select-none touch-callout-none` + preventDefault |

### Spring 动画统一参数

全应用所有页面切换使用统一的弹簧物理参数：
```typescript
transition={{ type: 'spring', stiffness: 150, damping: 20, mass: 0.8 }}
```
