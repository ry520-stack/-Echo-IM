# Echo 篇 5：星轨图片画廊 — 悬浮堆叠与 10 秒倒放

## 两种画廊模式

Echo 的动态系统"星轨"拥有两种图片展示模式。每条动态独立选择模式。

### 模式 1：悬浮循环堆叠 (FloatingStack)

**视觉效果**：最多同时显示 3 层图片。第 1 张 scale:1 居中，第 2 张 scale:0.92 右下偏移露出边缘，第 3 张及以后 scale:0.8 opacity:0 隐藏。

**手势**：仅顶层图片可拖拽 `drag="x"`。左滑超过 100px → 当前图片绕到数组末尾，第 2 张浮现。**右滑超过 100px → 回到上一张**。

**10 秒时空倒放**（核心难点）：

```typescript
useEffect(() => {
  if (offset > 0) {
    timerRef.current = setTimeout(() => {
      intervalRef.current = setInterval(() => {
        setActiveIndex(prev => {
          if (prev <= 1) { clearInterval(intervalRef.current!); return 0; }
          return prev - 1;
        });
      }, 300); // 每张 300ms
    }, 10000); // 10秒无操作
  }
  return () => { clearTimeout(timerRef.current); clearInterval(intervalRef.current); };
}, [activeIndex]);
```

### 模式 2：宫格横滑分页 (PaginatedGrid)

每页 9 张 `grid-cols-3`，超过 9 张自动分页。整体 `drag="x"` 横滑翻页，底部 `1/2` 页码指示器。封面图右上角 ⊞ 图标，点击展开宫格。

## 自定义封面

上传图片后可通过"设封面"按钮指定任意一张为封面。Gallery 模式（stack/grid）和封面索引（coverIndex）存储在 `localStorage` 中，每条动态独立。

## 全屏 Lightbox

点击图片进入全屏预览，支持左右滑切换。使用 `AnimatePresence` + `spring(150,20,0.8)` 过渡，左上角 `2/5` 页码，右下角保存按钮。

## 踩坑

- **AnimatePresence height 动画残留**：宫格展开使用 `height: 0 → auto` 动画退出时 DOM 占位撑爆卡片。改为纯 `opacity` 淡入。
- **图片白屏**：FloatingStack 的 key 用 `{url}` 而非 `{index}`，避免数组重排时渲染异常。
- **画廊模式全局污染**：早期用全局 `galleryMode` 状态，修改一条影响所有动态。改为 `{ [momentId]: 'stack'|'grid' }` 独立存储。
